# App Store Connect 設定

## 概要

App Store Connectでアプリを登録し、審査に必要な情報を入力します。

**所要時間**: 20-30分

## 前提条件

- [ ] Apple Developer Program に登録済み
- [ ] プライバシーポリシーのURL準備済み
- [ ] スクリーンショット準備済み（または同時進行）

## 手順

### 1. App Store Connect にログイン

1. [App Store Connect](https://appstoreconnect.apple.com/) にアクセス
2. Apple ID でサインイン
3. 「マイApp」を選択

### 2. 新規アプリの作成

#### 2.1 基本情報の入力

1. 左上の **「+」** → **「新規App」** をクリック

2. **プラットフォーム**: iOS

3. **名前**: `Spark Vault`
   - この名前がApp Storeに表示されます
   - 最大30文字

4. **プライマリ言語**: 日本語

5. **バンドルID**: `com.ogadix.sparkvault`
   - ドロップダウンから選択
   - 表示されない場合は、Apple Developer で Identifier を登録

6. **SKU**: `spark-vault-001`
   - 内部管理用の一意の識別子
   - 変更不可なので慎重に

7. **ユーザーアクセス**: フルアクセス

8. **作成** をクリック

### 3. App情報の設定

#### 3.1 基本情報

**App情報** タブで入力:

1. **プライバシーポリシーURL**:
   ```
   https://gaku52.github.io/spark-vault/privacy-policy.html
   ```
   *(実際のURLに置き換えてください)*

2. **カテゴリ**:
   - **プライマリ**: 仕事効率化
   - **セカンダリ**: ビジネス（オプション）

3. **サブタイトル** (オプション、最大30文字):
   ```
   アイデアと目標を一元管理
   ```

4. **ライセンス契約**: Appleの標準使用許諾契約

5. **保存**

#### 3.2 年齢制限

1. **年齢制限** セクションで **「編集」**

2. すべての項目で **「なし」** または **「まれ/軽度」** を選択
   - 暴力的な内容: なし
   - 性的な内容: なし
   - 賭博: なし
   - 医療情報: なし

3. **完了**

### 4. 価格および配信状況

#### 4.1 価格設定

1. **価格および配信状況** タブを選択

2. **価格**: 無料 (0円)

3. **配信可能な国または地域**:
   - **すべて** を選択
   - または特定の国を選択（日本、アメリカなど）

4. **保存**

### 5. App プライバシー

#### 5.1 プライバシーに関する質問

1. **App プライバシー** セクションで **「開始」**

2. **データ収集**:
   - **データを収集しますか?**: はい

3. **データの種類**:

   **連絡先情報**:
   - [ ] メールアドレス
     - **収集目的**: アカウント管理、カスタマーサポート
     - **第三者と共有**: いいえ
     - **トラッキングに使用**: いいえ
     - **ユーザーにリンク**: はい

   **ユーザーコンテンツ**:
   - [ ] 写真またはビデオ（画像添付機能がある場合）
     - **収集目的**: アプリ機能
     - **第三者と共有**: いいえ
     - **トラッキングに使用**: いいえ
     - **ユーザーにリンク**: はい

   - [ ] その他のユーザーコンテンツ（アイデアのテキスト）
     - **収集目的**: アプリ機能
     - **第三者と共有**: いいえ
     - **トラッキングに使用**: いいえ
     - **ユーザーにリンク**: はい

4. **公開**

### 6. 最初のバージョンの準備

#### 6.1 バージョン情報

1. 左側のメニューで **「iOS App」** → **「1.0 リリースの準備」** を選択

2. **新機能**（最大4000文字、日本語）:
   ```
   初回リリース

   【主な機能】
   • アイデアを即座に記録
   • カテゴリとタグで整理
   • 検索機能で素早くアクセス
   • 目標設定と進捗管理
   • クラウド同期で安全に保存
   • シンプルで使いやすいUI

   Spark Vaultは、あなたのアイデアと目標を一元管理するアプリです。
   思いついたアイデアを素早く記録し、カテゴリやタグで整理できます。
   ```

3. **プロモーションテキスト** (オプション、最大170文字):
   ```
   アイデアを即座に記録。目標を管理。すべてを一つの場所で。
   ```

### 8. アプリレビュー情報

#### 8.1 連絡先情報

**審査担当者への連絡先**:
- **名前**: あなたの名前
- **電話番号**: 日本の携帯番号 (例: +81-90-1234-5678)
- **メールアドレス**: 連絡可能なメールアドレス

#### 8.2 デモアカウント（必須）

審査担当者がアプリをテストするためのアカウント:

**デモアカウント情報**:
- **ユーザー名**: `review@example.com` （実際に作成したアカウント）
- **パスワード**: 簡単なパスワード（例: `Review123!`）

**注意**: 審査前にこのアカウントを実際に作成しておく

## よくある問題と対処法

### 問題1: Bundle IDが表示されない

**原因**: Apple Developerで Identifier が登録されていない

**対処法**:
1. [Apple Developer](https://developer.apple.com/) にログイン
2. `Certificates, Identifiers & Profiles` → `Identifiers`
3. `+` → `App IDs` を選択
4. Bundle ID: `com.ogadix.sparkvault` を登録

### 問題2: プライバシーポリシーURLが無効

**原因**: URLが公開されていない、またはHTTPSでない

**対処法**:
1. URLにブラウザでアクセスして確認
2. HTTPSであることを確認
3. モバイルでも表示されることを確認

### 問題3: スクリーンショットのサイズが合わない

**原因**: デバイスサイズが正確でない

**対処法**:
→ [05-screenshot-guide.md](./05-screenshot-guide.md) で正確なサイズを確認

# Spark Vault

ひらめき（Spark）を保管庫（Vault）に - アイデアを即座に記録し、自己成長を多角的に管理するWebアプリケーション

## プロジェクト概要

**Spark Vault**は、思いついたアイデアを瞬時にメモし、DJ活動、エンジニアリング、健康管理、思考パターンなど、人生の様々な側面での成長を一元管理するためのアプリケーションです。

### 主な目的

- IQの波を一定に保ち、生産性を向上
- 自己認識を深め、年収向上に貢献
- 継続的な自己改善サイクルの確立

## 主要機能

### 1. アイデアメモ
思いついたアイデアを即座に記録し、カテゴリやタグで整理

### 2. DJ成長記録
- スキルトラッキング（ミキシング、機材操作、選曲など）
- ミックス練習記録
- パフォーマンスメモ

### 3. エンジニア自己管理
- 技術学習記録
- プロジェクト進捗管理
- スキルマトリックス

### 4. 健康管理
- 睡眠・体調記録
- 運動トラッキング
- 食事ログ

### 5. 思考法記録
- 思考パターン分析
- 意思決定ログ
- IQ変動トラッキング

## 技術スタック

- **フロントエンド**: React 18 + TypeScript + Vite
- **UI**: TailwindCSS + shadcn/ui
- **ルーティング**: React Router
- **フォーム**: React Hook Form + Zod
- **データ可視化**: Recharts
- **バックエンド**: Supabase (PostgreSQL)
- **認証**: Supabase Auth
- **デプロイ**: Vercel

```bash
# 依存関係のインストール
npm install

# 環境変数設定
cp .env.example .env.local
# .env.local にSupabaseのURLとキーを設定

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
